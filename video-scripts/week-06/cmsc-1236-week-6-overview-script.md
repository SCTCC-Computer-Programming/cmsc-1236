## What You'll Learn This Week

The program you'll be working with is already partially built. The data, display functions, and `main()` are all in place. Five core functions are defined but empty. Your job is to write the function bodies so the program produces the correct output.

The key skill here is reading existing code to figure out what your functions need to do. The display functions show you what attributes and dictionary keys they expect. The data constants show you the structure of the input. The expected output on the assignment page shows you what the finished program should print. Between those three, the specification is complete. If you skip that investigation, your functions won't work.
